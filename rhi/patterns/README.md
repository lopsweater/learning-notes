# RHI 设计模式

本目录记录 RHI 层常见的设计模式和最佳实践。

## 文件说明

| 文件 | 内容 |
|------|------|
| `resource-pool.md` | 资源池模式、资源复用策略 |
| `descriptor-management.md` | 描述符管理、分配策略 |
| `state-tracking.md` | 资源状态追踪、自动屏障 |
| `command-buffer.md` | 命令缓冲区池化、多线程录制 |
| `frame-graph.md` | 帧图模式、资源生命周期管理 |
| `render-graph.md` | 渲染图模式、自动资源屏障 |
| `deferred-destruction.md` | 延迟销毁、GPU 同步销毁 |
| `upload-context.md` | 上传上下文、Staging Buffer |

## 常见模式概览

### 1. 资源管理模式

```
┌────────────────────────────────────────────────────────────┐
│                    资源生命周期                             │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  Create ──► Pool ──► Acquire ──► Use ──► Release ──► Pool │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

### 2. 描述符管理模式

```
┌────────────────────────────────────────────────────────────┐
│                   描述符分配策略                            │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  Shader Visible Heap (固定大小)                            │
│       │                                                    │
│       ├── Frame [0] Descriptors                            │
│       ├── Frame [1] Descriptors                            │
│       └── Frame [2] Descriptors                            │
│                                                            │
│  Non-Shader Visible Heap (动态分配)                        │
│       │                                                    │
│       └── CPU Staging Area                                 │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

### 3. 命令提交模式

```
┌────────────────────────────────────────────────────────────┐
│                    多线程命令录制                           │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  Thread 0: CommandList[0] ──┐                              │
│  Thread 1: CommandList[1] ──┼──► Submit ──► GPU            │
│  Thread 2: CommandList[2] ──┘                              │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

### 4. 同步模式

```
┌────────────────────────────────────────────────────────────┐
│                    帧同步模型                               │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  CPU: Frame[N] ──► Frame[N+1] ──► Frame[N+2]              │
│         │              │              │                    │
│         ▼              ▼              ▼                    │
│  GPU: Frame[N] ──► Frame[N+1] ──► Frame[N+2]              │
│                                                            │
│  Fence[N] ──────────► Wait ──────────► Signal              │
│                                                            │
└────────────────────────────────────────────────────────────┘
```
