## 什么是 RHI？

RHI（Render Hardware Interface）是游戏引擎中的**渲染硬件抽象层**，它：

- **封装底层图形 API**（D3D12、Vulkan、Metal）
- **提供统一接口**给上层渲染器使用
- **隐藏平台差异**，实现跨平台渲染
- **优化性能**，充分利用硬件特性

## 为什么需要 RHI？

| 问题 | RHI 解决方案 |
|------|-------------|
| 不同 API 差异大 | 统一抽象接口 |
| 平台特性不同 | 特性检测 + 回退机制 |
| 资源管理复杂 | 统一资源生命周期管理 |
| 同步模型不同 | 统一同步原语抽象 |

